To use these elements, include flottplot-extras-min.js and flottplot-extras.css in addition to the Flottplot core: 

<link rel="stylesheet" type="text/css" href="path/to/flottplot-extras.css">
<script type="text/javascript" src="/path/to/flottplot-extras-min.js"></script>

Elements

Cursors

Mirror the cursor position in multiple elements.

Definition

<fp-cursors id="ID">
  <fp-cursor target="TARGET" cursor="CURSOR" class="CLASS" style="STYLE"></fp-cursor>
  [...]
</fp-cursors>
ID (optional) own ID.
TARGET ID of the element included in the cursor group.
CURSOR (optional) cursor type.
  • pointer (default): arrow-like pointer.
  • hline: horizontal line with width of target element.
  • vline: vertical line with height of target element.
  • hidden: no mirrored cursor is shown for the target, but the mouse position is still broadcast to the other elements in the cursor group.
CLASS (optional) override for the default class name of the mirrored cursor.
STYLE (optional) override for the default style of the mirrored cursor. If a custom class is specified to style the cursor, set to empty to remove the default styling which might otherwise override the style specified in the class.

<fp-cursor> elements should be placed directly into the body of the document as they use absolute positioning to place the additional cursors.

Fullscreen Overlay

Fullscreen overlay when clicking on an image (e.g. <fp-plot> or <fp-stack>). In the overlay, the plot will have its native resolution or is scaled to the maximum possible size that fits the browser's viewport if the native size is larger than the viewport. Click anywhere to close the overlay again.

Definition

<fp-overlay id="ID"></fp-overlay>
ID own ID.

Actions

show show the overlay. Only shows if an element has previously placed content into the overlay.
hide hide the overlay.
toggle show the overlay if currently hidden, hide if currently shown.

Plot Stack

Display multiple <fp-plot> or <img> elements on top of each other.

Definition

<fp-stack>
  <fp-plot src="..."></fp-plot>
  <img src="...">
  [...]
</fp-stack>
