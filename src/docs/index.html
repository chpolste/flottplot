Overview

Flottplot defines custom HTML tags and converts them into control and other elements that interact with each other. The primary use case of Flottplot is to display and browse large collections of images in the browser. Multiple images can be placed on a single page and navigated in a synchronized fashion. Development of Flottplot mainly targets the organization of plots containing meteorological data. Note that Flottplot does not create new images, it just arranges images already plotted elsewhere.

Downloads

Choose the file(s) that are right for you:

  • flottplot.css: Additional styling required for some elements.
