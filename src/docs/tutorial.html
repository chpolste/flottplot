Initializing

Download Flottplot scan.

Copy the HTML template into a new file test.html.

<!doctype html>
<html>
<head>
  <meta charset="utf-8">
  <title>A Basic Flottplot Page</title>
  <script type="text/javascript" src="flottplot-scan-min.js"></script>
</head>
<body>

</body>
</html>

Open the HTML file with a browser.

With the Flottplot Python package, a new page test.html as shown above can be initialized more easily by running

python3 -m flottplot init test.html

on the command line.
